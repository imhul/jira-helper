# Jira Helper

**Jira Helper** - is a simple tool designed to help you manage your Jira tickets more efficiently. All changes are saved automatically every time you update any field, so there is no separate save button to worry about. When you add a new ticket, the most important step is filling in the **Ticket ID** and **Ticket Title** in the form. The remaining fields are secondary and can be completed later if you do not have all the details yet. You can see the open source code of this app [here](https://github.com/imhul/jira-helper).
